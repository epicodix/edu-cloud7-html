HTML 문서의 기본 구조

HTML 문서는 웹 페이지의 뼈대를 구성하며, 웹 브라우저가 내용을 올바르게 해석하고 표시할 수 있도록 일정한 구조를 가집니다. 모든 HTML 문서는 `<!DOCTYPE html>` 선언으로 시작하여 문서 유형을 정의하고, `<html>` 태그로 전체 문서를 감쌉니다.

`<html>` 태그 안에는 크게 두 부분으로 나뉩니다: `<head>`와 `<body>`.

<head> 태그

`<head>` 태그는 웹 페이지의 메타데이터(화면에 직접 표시되지 않는 정보)를 포함합니다. 여기에는 문서의 제목, 문자 인코딩, 외부 스타일시트 연결, 스크립트 파일 연결 등 브라우저에 필요한 설정 정보가 들어갑니다.

  • `<meta charset="UTF-8">`: 문서의 문자 인코딩을 지정합니다. 한글 등 다양한 문자를 올바르게 표시하기 위해 중요합니다.
  • `<meta name="viewport" content="width=device-width, initial-scale=1.0">`: 모바일 기기에서 웹 페이지가 올바르게 반응형으로 표시되도록 설정합니다.
  • `<title>`: 웹 브라우저 탭에 표시될 문서의 제목을 정의합니다.

<body> 태그

`<body>` 태그는 웹 페이지에 실제로 표시될 모든 콘텐츠를 포함합니다. 텍스트, 이미지, 링크, 비디오 등 사용자가 볼 수 있는 모든 요소가 이 안에 작성됩니다.

주요 HTML 태그: 제목과 단락

제목 태그 (<h1> ~ <h6>)

HTML에서 제목은 `<h1>`부터 `<h6>`까지의 태그를 사용하여 정의합니다. `<h1>`이 가장 중요한 최상위 제목이며, `<h6>`로 갈수록 중요도가 낮아집니다. 이 태그들은 문서의 구조와 계층을 나타내며, 검색 엔진 최적화(SEO)에도 중요한 역할을 합니다. 

<h1>가장 큰 제목</h1>
<h2>두 번째 큰 제목</h2>
<h3>세 번째 큰 제목</h3>
<h4>네 번째 큰 제목</h4>
<h5>다섯 번째 큰 제목</h5>
<h6>가장 작은 제목</h6>

단락 태그 (<p>)

`<p>` 태그는 웹 페이지에서 텍스트 단락(paragraph)을 정의할 때 사용합니다. 브라우저는 `<p>` 태그의 시작과 끝에 자동으로 여백을 추가하여 단락을 구분합니다. 일반적인 텍스트 콘텐츠를 담는 데 가장 많이 사용되는 태그 중 하나입니다. 

<p>이것은 첫 번째 단락입니다. 여기에 웹 페이지의 주요 텍스트 콘텐츠가 들어갑니다.</p>
<p>이것은 두 번째 단락입니다. 각 단락은 <p> 태그로 구분됩니다.</p>
