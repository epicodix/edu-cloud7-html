# Django 프로젝트 시리즈

이 폴더는 Django 프로젝트를 단계별, 주제별로 나누어 관리하기 위해 만들어졌습니다.

## 폴더 구조

### 📂 01_start
Django 프로젝트의 가장 기본적인 설정을 다루거나, 새로운 프로젝트를 시작할 때 사용하는 공간입니다.

### 📂 02_web
본격적인 웹 애플리케이션의 모델(Model), 뷰(View), 템플릿(Template) 등 핵심 기능을 구현하는 프로젝트를 보관합니다.

### 📂 03_nginx
Django 프로젝트를 실제 서버에 배포하는 과정을 다룹니다. Nginx, Gunicorn 등 웹 서버 연동 관련 프로젝트를 보관합니다.
