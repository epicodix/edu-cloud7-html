CSS Flexbox와 Grid 레이아웃

Flexbox와 Grid는 복잡한 웹 레이아웃을 쉽고 효율적으로 구성할 수 있게 해주는 최신 CSS 레이아웃 모델입니다.

Flexbox 레이아웃

Flexbox(Flexible Box)는 주로 1차원(한 줄 또는 한 열) 레이아웃을 위해 설계되었습니다. 아이템들을 축(axis)에 따라 정렬하고 공간을 분배하는 강력한 방법을 제공합니다.

Flexbox 예제

1 (grow: 1)
2 (grow: 2)
3

핵심 속성 (Container)

  • display: flex;: Flexbox 레이아웃을 시작합니다.
  • flex-direction: 아이템이 배치될 주 축의 방향을 설정합니다 (row, column 등).
  • justify-content: 주 축을 기준으로 아이템을 정렬합니다 (flex-start, center, space-between 등).
  • align-items: 교차 축을 기준으로 아이템을 정렬합니다 (stretch, center, flex-start 등).

핵심 속성 (Item)

  • flex-grow: 아이템이 컨테이너 내 여백을 차지하는 비율을 지정합니다.
  • flex-shrink: 컨테이너 공간이 부족할 때 아이템이 줄어드는 비율을 지정합니다.
  • flex-basis: 아이템의 기본 크기를 지정합니다.

Grid 레이아웃

Grid는 2차원(행과 열) 레이아웃 시스템으로, Flexbox보다 더 복잡한 구조를 만들 때 유용합니다. 웹 페이지의 전체적인 틀을 잡는 데 매우 효과적입니다.

Grid 예제

1
2
3
4
5

핵심 속성 (Container)

  • display: grid;: Grid 레이아웃을 시작합니다.
  • grid-template-columns: 그리드의 열(column) 트랙과 크기를 정의합니다. (예: 1fr 1fr 1fr, 100px auto 100px)
  • grid-template-rows: 그리드의 행(row) 트랙과 크기를 정의합니다.
  • gap: 그리드 아이템 사이의 간격을 설정합니다.

핵심 속성 (Item)

  • grid-column: 아이템이 어떤 열 라인에서 시작하고 끝날지 지정합니다.
  • grid-row: 아이템이 어떤 행 라인에서 시작하고 끝날지 지정합니다.

Flexbox vs. Grid

두 레이아웃 모델은 상호 보완적이며, 함께 사용될 때 가장 강력합니다.

  • Flexbox 사용시기: 컴포넌트 내부의 아이템들을 한 줄(또는 한 열)로 정렬할 때. 예를 들어, 네비게이션 바, 버튼 그룹, 카드 내의 콘텐츠 정렬에 적합합니다.
  • Grid 사용시기: 전체 페이지의 레이아웃이나 복잡한 2차원 구조를 만들 때. 예를 들어, 헤더, 사이드바, 메인 콘텐츠, 푸터로 구성된 페이지의 큰 틀을 잡는 데 적합합니다.
