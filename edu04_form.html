HTML 폼 요소: Button, Select, Fieldset & Web API

<button> 태그

<button> 태그는 클릭 가능한 버튼을 생성합니다. 사용자의 액션에 반응하여 폼을 제출하거나 스크립트를 실행하는 데 사용됩니다.

버튼 유형 (type 속성)

  • button: 기본 동작이 없는 일반 버튼입니다. JavaScript와 함께 사용됩니다.
  • submit: 폼 데이터를 서버로 제출하는 버튼입니다.
  • reset: 폼의 모든 입력 필드를 초기값으로 되돌립니다.

버튼 예제


예제 HTML 코드

<!-- 폼 제출 및 초기화 버튼 -->
<form action="#">
    <label for="username-button">사용자명:</label>
    <input type="text" id="username-button" name="username">
    <button type="submit">제출</button>
    <button type="reset">초기화</button>
</form>

<!-- 일반 스크립트 실행 버튼 -->
<button type="button" onclick="alert('일반 버튼 클릭!')">일반 버튼</button>

<select> 태그

<select> 태그는 드롭다운 목록을 생성합니다. 사용자가 여러 옵션 중 하나 또는 여러 개를 선택할 수 있도록 합니다.

주요 속성

  • multiple: 여러 옵션을 동시에 선택할 수 있도록 허용합니다.
  • size: 한 번에 보여줄 옵션의 개수를 지정합니다.

Select 예제



예제 HTML 코드

<!-- 단일 선택 드롭다운 -->
<label for="framework-select">선호하는 프레임워크:</label>
<select id="framework-select" name="frameworks">
    <option value="">선택하세요...</option>
    <option value="React">React</option>
    <option value="Vue">Vue.js</option>
    <option value="Angular">Angular</option>
</select>

<!-- 다중 선택 리스트 -->
<label for="fruits-select">좋아하는 과일 (다중 선택 가능):</label>
<select id="fruits-select" name="fruits" multiple size="4">
    <option value="apple">사과</option>
    <option value="banana">바나나</option>
    <option value="orange">오렌지</option>
    <option value="grape">포도</option>
</select>

<fieldset> 태그

<fieldset> 태그는 폼(form) 안에서 관련된 요소들을 그룹화하는 데 사용됩니다. <legend> 태그를 함께 사용하여 그룹의 제목을 붙일 수 있습니다.

주요 특징

  • 폼의 구조를 시각적으로 명확하게 만들어줍니다.
  • 스크린 리더 사용자에게 폼의 각 섹션을 더 잘 이해할 수 있도록 돕습니다.
  • disabled 속성을 사용하여 그룹 전체의 요소들을 한 번에 비활성화할 수 있습니다.

Fieldset 예제

로그인 정보

예제 HTML 코드

<form>
    <fieldset>
        <legend>로그인 정보</legend>
        <label for="username-fieldset">사용자 ID:</label>
        <input type="text" id="username-fieldset" name="username">
        <label for="password-fieldset">비밀번호:</label>
        <input type="password" id="password-fieldset" name="password">
    </fieldset>
</form>

웹 API (Web API) 소개

API(Application Programming Interface)는 애플리케이션 간의 상호작용을 위한 인터페이스입니다. 웹 API를 통해 JavaScript는 브라우저의 기능(예: Geolocation)을 사용하거나, 서버와 데이터를 교환(예: Fetch API)할 수 있습니다.

주요 웹 API

  • DOM API: HTML 문서를 제어합니다.
  • Fetch API: 서버와 비동기 통신을 합니다.
  • Geolocation API: 사용자의 위치 정보를 가져옵니다.
  • Canvas API: 2D 그래픽을 그립니다.
  • Web Storage API: 브라우저에 데이터를 저장합니다.

Fetch API 예제

아래 버튼을 클릭하면, 공개 API에서 사용자 데이터를 가져와 콘솔에 출력합니다.

예제 JavaScript 코드

function handleFetchUserData() {
    fetch('https://jsonplaceholder.typicode.com/users/1')
        .then(response => {
            if (!response.ok) {
                throw new Error('네트워크 응답 오류');
            }
            return response.json();
        })
        .then(user => {
            console.log('가져온 사용자 데이터:', user);
            alert(user.name + '님의 데이터를 콘솔에서 확인하세요.');
        })
        .catch(error => {
            console.error('데이터 fetch 중 오류 발생:', error);
            alert('데이터를 가져오는 데 실패했습니다.');
        });
}
